JQL( Javascript Query Language  )
=================================

* Javascript에서 SQL 문법을 사용하여 JSON 데이터를 조작할 수 있는 라이브러리 입니다.
* 실제로 DB에 접속하거나 하지는 않습니다.
* 단지 서버와 클라이언트 사이의 데이터를 JSON으로 주고 받을 때, 또는 화면에서 데이터를 조작하여 보여주어야 할 때, 편하게 작업하는 용도로 사용됩니다.
